## Title
A simple personal diary demo web page designed to showcase a clean and minimal user interface. This project focuses on layout design, typography, and visual structuring using HTML and CSS.

It was built as a UI-focused demo to practice front-end layout and styling fundamentals.

## Features
- Clean and minimal diary-style layout
- Structured sections for writing entries
- Simple, distraction-free design
- Responsive layout for different screen sizes

## Tech Stack
- HTML
- CSS

## Project Structure
index.html – diary page structure

style.css – layout, styling, and typography

## What I Learned
- Structuring web pages using semantic HTML
- Styling layouts with CSS
- Improving visual hierarchy and readability
- Designing simple UI-focused demo pages
